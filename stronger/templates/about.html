Objectives

The driving concepts behind stronger are to:

  • offer a free workout tracking tool for people who enjoy weight training
  • deliver a greater user experience than similar pre-existing products
  • instigate progress through healthy competition
  • encourage users to become involved in the development of the software

Platform Stack

Stronger uses a range of Open Source technolgies and libraries including:

  • Python
  • Django
  • PostgreSQL
  • jQuery
  • HighChart
  • LESS
  • Twitter Bootstrap
  • Jenkins
  • Git
  • Apache HTTP Server

Get Involved

If there is a feature you would like to add or a bug you would like to fix, please check out the source code on github and submit a patch.
